Getting Started
---------------

### Stack
 * MariaDB
 * Nginx
 * Redis
 * PHP
 * RabbitMQ
 * ElasticSearch
 * Mailhog
 * Node


### CLI Commands
 * `bin/project start` - starts the project
 * `bin/project stop` - stops the project
